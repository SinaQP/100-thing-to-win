# 100 Things to Win

Offline-first Android-focused Flutter habit tracker.

## Phase 1 Scope
- Clean architecture project skeleton
- App-wide theme system (light/dark/system)
- Router + bottom navigation shell
- Placeholder feature screens
- Offline persistence and testing foundations

## Run
1. Install Flutter SDK (stable)
2. Run:
   - `flutter pub get`
   - `flutter run`

## Structure
See `lib/` for feature-first clean architecture modules.

## Next
Phase 2 implements full habit CRUD, daily check-ins, streak logic, local DB integration, and polished UI states.
